Manual Upgrade dos Ambientes de Produção de outra versão para outra versão que não se encontra 
==============================================================================================
na lista de manuais para upgrade
================================

1. Apenas no servidor do monitoramento:
	 - Conectar como cronostech
     - No Windows Explorer alterar a opção para exibir arquivos de sistema temporariamente

2. Copiar o arquivo novo integr-fornecedor-x.y.z.jar do instalador para o servidor do fornecedor
 

3. Atualizar a chamada de integr-fornecedor-x.y.z.jar no Job15a15minOfertamentoJava_Windows.bat 
   no servidor do fornecedor
 
4. Apenas no servidor do monitoramento: 
     - No Windows Explorer voltar a opção para não exibir arquivos de sistema
