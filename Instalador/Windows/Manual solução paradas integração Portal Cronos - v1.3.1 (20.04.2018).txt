Possíveis causas de paradas da integração WinThor/PCronos:
==========================================================  
  - O servidor de vocês que está hospedando este serviço está desligado?

  - O disco rígido ("HD") C:\ está cheio?

  - Houve uma atualização de Windows?

  - Houve uma atualização de Java?

  - A memória RAM está 100% cheia ou perto disso?
    No caso que a memória RAM estiver mais de 95 %, favor deixar assim por enquanto 
    e não fazer nada ainda e avisar o contato "Eric Jo" via Skype (ou eric.jo@bol.com.br via email) 
    primeiro para conectar via AnyDesk/Team Viewer para testar uma solução automática para este problema.  
    (Isso é necessário apenas na primeira vez que isso acontece.) 

  - Algum anti-virus está travando a máquina? (100 % memória RAM)

  - O endereço IP, usuário ou senha do banco Oracle mudou?

  - A senha do usuário ws-jrembalagem foi alterado no site do Portal Cronos porém não foi atualizado
     no arquivo de configuração? (Este arquivo se encontra no diretório 
     C:\Arquivos de Programas PC\Integração Fornecedor - Portal Cronos\conf\Integração Fornecedor - Portal Cronos.properties) 

  - Outras ideias suas. 

Favor NÃO SIMPLESMENTE REINICIAR O SERVIDOR, porém identificar a causa para podermos evitar repetição no futuro. 

No caso que você mesmo consegue resolver o problema, favor verificar após 15 minutos se as cotações pendentes foram ofertadas automaticamente. 
Se você não tiver um usuário/senha no site do Portal Cronos, favor verificar isso com os vendedores. 
 
No caso que as ofertas automáticas ainda não voltaram a funcionar, 
veja as outras possíveis causas neste manual. 
 
Apenas no último caso, se não conseguir resolver, 
entrar em contato com o contato "Eric Jo" via Skype (ou com eric.jo@bol.com.br via email), 
e colocar o AnyDesk/Team Viewer no ar, e informar o ID e a senha via Skype, 
para ele ver se a causa foi alguma falha dentro do serviço das ofertas automáticas.

  
Outras dicas:
=============
- O servidor de hospedagem do serviço das ofertas automáticas 
  é o servidor no qual existe um diretório C:\Arquivos de Programas PC\ 
             
- Nunca precisa reiniciar ou startar o serviço. Ao ligar ou reiniciar o servidor, 
  o serviço starta automaticamente.
