Manual Upgrade dos Ambientes de Produção de 2.5.2 para 2.6
==========================================================

1. PRIMEIRO, antes de alterar a versão do .jar e antes de renomear Job15a15minOfertamentoJava_Windows.bat: 
   no Windows Task Scheduler > Propriedades > Configurações: alterar o último combobox 
   de "Não iniciar uma nova instância" para "Interromper a instância existente"
   
   No caso que o processo JRE estiver travado, ficar acompanhando se a próxima execução do Scheduler 
   derruba isso automaticamente. (Se alterar a versão do .jar ou a data/hora de
   "Job15a15minOfertamentoJava_Windows.bat" primeiro, parece que isso não funciona.) 
  
2. Renomear Job15a15minOfertamentoJava_Windows.bat para xJob15a15minOfertamentoJava_Windows.bat

3. Copiar o arquivo novo integr-fornecedor-2.6.jar do instalador para o servidor do fornecedor
 
4. Atualizar a chamada de integr-fornecedor-2.6.jar no Job15a15minOfertamentoJava_Windows.bat 
   no servidor do fornecedor
 
5. Renomear xJob15a15minOfertamentoJava_Windows.bat de volta para Job15a15minOfertamentoJava_Windows.bat

6. Aproveitar atualizar espaço livre em Tabela Fornecedores_Versões.txt
