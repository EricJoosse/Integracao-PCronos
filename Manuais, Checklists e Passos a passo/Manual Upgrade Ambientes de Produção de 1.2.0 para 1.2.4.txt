Manual Upgrade dos Ambientes de Produção de 1.2.0 para 1.2.4
============================================================

1. Copiar arquivo novo integr-fornecedor-1.2.4.jar do instalador para o servidor do fornecedor
   Apagar a versão anterior.
 
2. Atualizar a chamada de integr-fornecedor-1.2.4.jar no Job15a15minOfertamentoJava_Windows.bat 
   no servidor do fornecedor
 
3. Copiar o arquivo sapjco3.jar do instalador para o servidor do fornecedor (dir /lib/)
   apenas para evitar runtime error que está faltando de pacotes e classes, mesmo que a 
   integração é com outro sistema diferente do SAP.
    
   Obs.: os arquivos sapjco3.dll e sapjco3.pdb apenas precisam ser copiados 
         no caso de integração com o SAP
