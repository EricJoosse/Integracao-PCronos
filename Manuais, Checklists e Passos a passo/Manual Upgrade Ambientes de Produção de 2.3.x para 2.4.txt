Manual Upgrade dos Ambientes de Produção de 2.3.x para 2.4
==========================================================

1. Adicionar "exit" seguida por uma linha em branco no início do arquivo Job15a15minOfertamentoJava_Windows.bat
   no servidor do fornecedor. Não renomear Job15a15minOfertamentoJava_Windows.bat para evitar que o Scheduler 
   possivelmente se perde comprometendo os derrubamentos automáticos dos JRE´s travados

   
2. Copiar o arquivo novo integr-fornecedor-2.4.jar do instalador para o servidor do fornecedor
 

3. Atualizar a chamada de integr-fornecedor-2.4.jar no arquivo Job15a15minOfertamentoJava_Windows.bat 
   no servidor do fornecedor
 

4. - criar o diretório novo C:/ProgramData/PortalCronos/Logs/Local (não precisa dar acesso a ninguêm)

   - mover o conteúdo de C:/temp/PortalCronos/XML para C:/ProgramData/PortalCronos/Logs/Local

   - apagar o diretório antigo C:/temp/PortalCronos/XML

   - no arquivo /conf/Integração Fornecedor - Portal Cronos.properties substituir:

DiretorioArquivosXml              = C:/temp/PortalCronos/XML
#                                          # Opções PADRÃO : 
#                                              - no caso de Windows : C:/temp/PortalCronos/XML
#                                              - no caso de Linux   : /var/PortalCronos/XML
   
     por:
DiretorioArquivosXml              = C:/ProgramData/PortalCronos/Logs/Local
#                                          # Opções PADRÃO: 
#                                              - no caso de Windows : C:/ProgramData/PortalCronos/Logs/Local
#                                              - no caso de Linux   : /var/PortalCronos/Logs/Local

5. Remover "exit" e a linha em branco no início do arquivo Job15a15minOfertamentoJava_Windows.bat
   no servidor do fornecedor
