Manual Upgrade dos Ambientes de Produção de 1.3.0 para 2.0.0
============================================================

1. Copiar arquivo novo integr-fornecedor-2.0.0.jar do instalador para o servidor do fornecedor
   Apagar a versão anterior.
 
2. Atualizar a chamada de integr-fornecedor-2.0.0.jar no Job15a15minOfertamentoJava_Windows.bat 
   no servidor do fornecedor
 
3. Copiar os arquivos mssql-jdbc-6.2.2.jre8.jar e mail-1.5.0-b01.jar do instalador 
   para o servidor do fornecedor (dir /lib/), 
   apenas para evitar runtime error que está faltando de pacotes e classes, mesmo que a 
   integração não usa SQL Server ou email.
